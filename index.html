LombardPress is the publishing component of a digital ecosystem designed to help faciliate a new kind of editing and publication of historical texts.

The primary focus of LombardPress is to faciliate the publication, dissemination, and study of scholastic commentaries and texts. LombardPress applications are designed to use and publish information made available via emerging scholarly standards and API. In particular, we rely on the Scholastic Commentaries and Text Archive (SCTA) ontology and database and the International Image Interoperability Framework API. Because these applications are built to understand common standards, these publication resources are designed to be resuable for any data set that follows the same standards for data creation and dissimenation. For more information see our about page.
