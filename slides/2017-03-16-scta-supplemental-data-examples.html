Publishing a Layer of Transcriptions

A layer example anouncing an a transcriptions for each page/canvas as an annotations list.

http://scta.info/iiif/rothwellcommentary/penn/layer/transcription

Notice the addition of the "sc:forCanvas" property. This was necessary to allow the easy association of an annotationList with a given canvas in a manifest
